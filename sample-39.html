使用说明

1.引入CSS和JS文件 
<link href="css/codeboard.css" rel="stylesheet" type="text/css">
<!--代码部分-->
<script src="js/jquery.min.js"></script>
<script src="js/codeboard.min.js"></script>
2.调用 
<pre class="codeblock">/*代码片段*/</pre>

格式：<pre class="codeblock" start="6" line="7,9">
start:起始行号（缺省为1）
line:需要高亮的行号（缺省为不高亮）

示例（从第6行开始，7和9行高亮）： 
public class HelloWorld(){	
	public static void main(String[] args)
	{
		System.out.print("Hello World!");		
	}	
}
3.其他

最多支持行数为9999,超出范围需要调整.linenum的宽度。
Wordpress等博客中引用需覆盖原来的样式。
